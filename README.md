# UltimateBoxingSimulator

Опис проєкту

Ultimate Boxing Simulator — це віконна гра-симулятор боксу, розроблена на C# із використанням фреймворку WPF. Гра дозволяє користувачам брати участь у віртуальних боксерських поєдинках, де переможець визначається на основі суми атрибутів персонажів (сила, витривалість, швидкість, рефлекси, техніка) із певними коефіцієнтами. Інтерфейс гри побудований на кнопках (наприклад, «У бій»), без використання 2D чи 3D графіки. Проєкт демонструє застосування об’єктно-орієнтованого програмування (ООП) та шаблонів проєктування.

Використані технології:
Мова програмування: C#
Фреймворк: WPF, .NET Framework
Інструменти: Visual Studio, Git
Шаблони проєктування: Singleton, Abstract factory, Prototype, Facade, Decorator, Proxy, Chain of responsibility, State, Strategy
Концепції: ООП, обробка подій, XAML для UI

Встановлення та запуск
Вимоги:
Visual Studio 2022 (або новіша) із підтримкою .NET Framework.
ОС Windows (для WPF-додатків).

Інструкція:
Склонуйте репозиторій: git clone [посилання на ваш репозиторій].
Відкрийте файл рішення (.sln) у Visual Studio.
Зберіть проєкт (Build Solution).
Запустіть додаток (F5 або Start Debugging).

Використання:
У головному вікні натисніть кнопку «У бій», щоб розпочати симуляцію поєдинку.
Переможець визначається автоматично на основі атрибутів персонажів.
